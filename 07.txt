7.A) Perform threshold operation

// Threshold Operation.
clc ;
a = imread ("");
a = rgb2gray (a);

t = input("Enter the threshold parameter ");
for i = 1:m
for j=1:n
if(a(i,j)<t)
b(i,j)=0;
else
b(i,j)=255;
end
end
end
figure(1)
imshow(uint8(a));

figure(2)
imshow(uint8(b));

xlabel(sprintf('Threshold value is %g' ,t));



7.B) Perform gray level slicing without background.
